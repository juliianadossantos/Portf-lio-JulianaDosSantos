Projeto independente

Projeto desenvolvido de forma totalmente independente com o objetivo de colocar em prática conceitos de front-end de maneira leve, criativa e divertida. Foi também uma oportunidade de tirar ideias do papel, permitindo que minha criatividade guiasse o desenvolvimento visual e estrutural do projeto.

Projeto Redes Sociais

Projeto tela login

Projeto desenvolvido para criar uma simulação de interface de tela de login com foco em validação de formulário e adaptação visual para diferentes dispositivos. A página inclui campos para email e senha, com validação básica que restringe o acesso quando informações obrigatórias não são preenchidas corretamente, proporcionando uma experiência mais realista mesmo sem backend.

O layout foi estruturado com HTML semântico e estilizado com CSS, utilizando Media Queries e breakpoints para tornar o design responsivo e adaptável a telas de diferentes tamanhos, com organização limpa e acessível do código.

Esse projeto demonstra minha capacidade de aplicar conceitos de front-end essenciais, trabalhar com formulários e responsividade, além de reforçar boas práticas de estruturação e estilização de interfaces web.

Tecnologias utilizadas: HTML • CSS • Responsividade com Media Queries • Validação visual de formulário •Git/GitHub
